## Теоретичні відомості

Лабораторна робота виконана мовою **TypeScript** з використанням середовища **Node.js** та інструменту **tsx** для запуску `.ts` файлів без попередньої компіляції.

### Структура програми на TypeScript

Програма на TypeScript складається з модулів — окремих файлів, кожен зі своїм призначенням. Модулі імпортуються в головний файл (`index.ts`), який є точкою входу. TypeScript розширює JavaScript статичною типізацією, що дозволяє виявляти помилки на етапі розробки.

### Організація виводу в консоль

Для виводу в консоль використовується об'єкт `console`:

```typescript
console.log("Hello, world!");           // вивід з переходом на новий рядок
console.log(`x=${x}; y=${y}`);          // шаблонні рядки (template literals)
console.log(`Зріст: ${h.toFixed(2)}`); // форматування чисел з плаваючою точкою
```

### Генератори та ітератори

У роботі активно використовуються **генератори** (`function*`) — спеціальні функції, що повертають ітератор і дозволяють покроково генерувати значення через ключове слово `yield`. Це забезпечує ліниве обчислення та ефективну роботу з послідовностями даних.

### Завдання 1 — Арифметичні операції над змінними

**Файл:** [`src/task1.ts`](src/task1.ts)

Створено 10 цілих змінних (`n1`–`n10`). Генератор `getNumbers()` послідовно повертає їх значення через `yield*`. Функція `reduceIterable` приймає ітерабельний об'єкт та функцію-редюсер і послідовно застосовує операцію до всіх елементів.

```typescript
const n1 = 10, n2 = 5, n3 = 2, n4 = 8, n5 = 4;
const n6 = 1, n7 = 3, n8 = 7, n9 = 6, n10 = 9;

function* getNumbers() {
  yield* [n1, n2, n3, n4, n5, n6, n7, n8, n9, n10];
}

const reduceIterable = (
  iterable: Iterable<number>,
  reducer: (a: number, b: number) => number,
  initial?: number
) => {
  const iterator = iterable[Symbol.iterator]();
  let result = initial !== undefined ? initial : iterator.next().value;
  for (const value of iterator) {
    result = reducer(result, value);
  }
  return result;
};

console.log(`Сума: ${reduceIterable(getNumbers(), (a, b) => a + b)}`);
console.log(`Різниця: ${reduceIterable(getNumbers(), (a, b) => a - b)}`);
console.log(`Добуток: ${reduceIterable(getNumbers(), (a, b) => a * b)}`);
console.log(`Частка: ${reduceIterable(getNumbers(), (a, b) => a / b)}`);
```

**Результат:**
```
Сума: 55
Різниця: -35
Добуток: 3628800
Частка: 0.00002755731922398589
```

### Завдання 3 — Клас User

**Файл:** [`src/task3.ts`](src/task3.ts)

Створено клас `User` з полями `id`, `age`, `name`, `surname`, `weight`, `height`. Генератор `getUsers()` створює 10 об'єктів класу. Допоміжні функції `mapIterable` (проєкція поля) та `sumIterable` (підсумок) обчислюють суми для кожного числового поля.

```typescript
class User {
  constructor(
    public id: number,
    public age: number,
    public name: string,
    public surname: string,
    public weight: number,
    public height: number,
  ) {}
}

function* getUsers() {
  yield new User(1, 22, "Олександр", "Петренко", 75.5, 180.2);
  yield new User(2, 30, "Марія", "Іваненко", 58.3, 165.0);
  // ... ще 8 об'єктів
}

const totalAge = sumIterable(mapIterable(getUsers(), (u) => u.age));
const totalWeight = sumIterable(mapIterable(getUsers(), (u) => u.weight));
const totalHeight = sumIterable(mapIterable(getUsers(), (u) => u.height));
```

**Результат:**
```

Сума ваги всіх користувачів: 709.5
```

### Завдання 5 — Реверс трьохзначного числа

**Файл:** [`src/task5.ts`](src/task5.ts)

Генератор `extractDigits` витягує цифри трьохзначного числа за допомогою операцій `%` (остача від ділення) та `Math.floor` (цілочисельне ділення). Генератор `reverseNumber` збирає цифри у зворотному порядку, множачи кожну на позиційний множник.

```typescript
const num: number = 412;

function* extractDigits(n: number) {
  yield n % 10;           // 2 (одиниці)
  yield Math.floor(n / 10) % 10;  // 1 (десятки)
  yield Math.floor(n / 100);      // 4 (сотні)
}

function* reverseNumber(n: number) {
  let multiplier = 100;
  for (const digit of extractDigits(n)) {
    yield digit * multiplier;
    multiplier /= 10;
  }
}

let reversed = 0;
for (const part of reverseNumber(num)) {
  reversed += part;
}
```

**Результат:**
```
Початкове число: 412
Реверсоване число: 214
```
