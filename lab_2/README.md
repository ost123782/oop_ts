### Умовний оператор if

Базова структура "розгалуження" забезпечує вибір одного з альтернативних шляхів алгоритму в залежності від перевірки деякої умови. В TypeScript реалізується за допомогою інструкції `if` / `else if` / `else`:

```typescript
if (condition1) {
  // виконується якщо condition1 — true
} else if (condition2) {
  // виконується якщо condition1 — false, condition2 — true
} else {
  // виконується якщо обидві умови — false
}
```

### Тернарний оператор

```typescript
const result = (condition) ? expressionTrue : expressionFalse;
```

### Оператор вибору switch

Оператор `switch` організує розгалуження по декількох напрямках. Кожна галузь відзначається константою і вибирається, якщо значення виразу збігається з цією константою:

```typescript
switch (expression) {
  case value1:
    // оператори
    break;
  case value2:
    // оператори
    break;
  default:
    // оператори за замовчуванням
}
```

### Завдання 4.1 — Обчислення f(x) (Варіант 6)

**Файл:** [`src/task1.ts`](src/task1.ts)

Використовуючи умовний оператор `if / else if / else`, реалізовано обчислення кусково-заданої функції `f(x)` для різних діапазонів значень `x`. Використано стандартні математичні функції з `Math`.

```typescript
function computeF(x: number): number {
  if (x <= 0.5) {
    return Math.abs(3 - Math.tan(x));
  } else if (x > 0.5 && x < 1) {
    return Math.pow(2 - Math.pow(x, 3), 1 / 5);
  } else {
    return Math.sin(x) - (1 / 3) * Math.pow(Math.cos(x), 5);
  }
}

const testValues = [-1, 0, 0.5, 0.7, 1, 2, 5];
for (const x of testValues) {
  console.log(`f(${x}) = ${computeF(x).toFixed(6)}`);
}
```

**Результат:**
```

f(0) = 3.000000
f(0.5) = 2.453698
f(0.7) = 1.106279
f(1) = 0.826123
f(2) = 0.913458
f(5) = -0.959536
```

### Завдання 4.2.3 — Кількість днів у місяці

За номером місяця визначається кількість днів. Для лютого враховується високосний рік за допомогою тернарного оператора.

```typescript
const monthNum: number = 2;
const isLeapYear: boolean = true;

switch (monthNum) {
  case 1: case 3: case 5: case 7: case 8: case 10: case 12:

    break;
  case 4: case 6: case 9: case 11:
    console.log(`Місяць ${monthNum} має 30 днів`);
    break;

    console.log(`Місяць ${monthNum} має ${days} днів (${isLeapYear ? "високосний" : "невисокосний"} рік)`);
    break;
}
```

**Результат:**
```
Місяць 2 має 29 днів (високосний рік)
```

### Завдання 4.2.4 — Текстове меню

**Файл:** [`src/task5.ts`](src/task5.ts)

Реалізовано інтерактивне текстове меню з використанням бібліотеки `prompt-sync` для введення даних з консолі та оператора `switch` для обробки вибору.

```typescript
import promptSync from 'prompt-sync';
const prompt = promptSync();

console.log("Меню:");
console.log("1 — Привітання");
console.log("2 — Запрошення до роботи");
console.log("3 — Завершити роботу");

const menuChoice: number = Number(prompt("Введіть номер меню (1-3): "));

switch (menuChoice) {
  case 1:
    console.log("Вітаємо вас! Раді бачити!");
    break;
  case 2:
    console.log("Запрошуємо вас до роботи на комп'ютері!");
    break;
  case 3:
    console.log("Дякуємо за роботу. До побачення!");
    break;
  default:
    console.log("Помилка: оберіть пункт від 1 до 3");
}
```

**Результат (при виборі 1):**
```
Вітаємо вас! Раді бачити!
```

### Завдання 4.2.7 — Калькулятор

**Файл:** [`src/task8.ts`](src/task8.ts)

Реалізовано текстове меню калькулятора з трьома операціями: добуток, сума, різниця. Вибір операції обробляється за допомогою `switch`.

```typescript
const a: number = 15;
const b: number = 7;
const operation: number = 1;

switch (operation) {
  case 1:

  case 2:
    console.log(`Сума: ${a} + ${b} = ${a + b}`);
    break;

    break;
  default:

}
```

**Результат:**
```
Добуток: 15 * 7 = 105
```
