## Теоретичні відомості

Циклічна композиція пропонує повторне виконання деякого внутрішнього оператора доти, поки існують умови для цього. Внутрішній оператор називають також тілом циклу.

### Цикл `for`

```typescript
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

### Цикл `while`

```typescript
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
```

### Завдання 1 — Вивести фразу 50 разів

**Файл:** [`src/task1.ts`](src/task1.ts)

Обрана фраза виводиться 50 разів спочатку циклом `for`, потім циклом `while`.

```typescript
const phrase = "Львівська Політехніка — найкращий університет!";

for (let i = 1; i <= 50; i++) {
  console.log(`${i}. ${phrase}`);
}

let j = 1;
while (j <= 50) {
  console.log(`${j}. ${phrase}`);
  j++;
}
```

### Завдання 5 — Табулювання функції (Варіант 6)

**Файл:** [`src/task5.ts`](src/task5.ts)

Функція `y = 2x·sin(x) - cos(x)` табулюється на проміжку [-2; 2] з кроком 0.2. Користувач вводить границі діапазону, програма підраховує кількість значень функції, що потрапляють у цей діапазон. Реалізовано двома способами: `for` та `while`.

```typescript
function f(x: number): number {
  return 2 * x * Math.sin(x) - Math.cos(x);
}

// Спосіб 1: цикл for
for (let x = a; x <= b; x += step) {
  const y = f(x);
  console.log(`${x.toFixed(1)}\t\t${y.toFixed(6)}`);
  if (y >= lowerBound && y <= upperBound) countFor++;
}

// Спосіб 2: цикл while
let x = a;
while (x <= b) {
  const y = f(x);
  console.log(`${x.toFixed(1)}\t\t${y.toFixed(6)}`);
  if (y >= lowerBound && y <= upperBound) countWhile++;
  x += step;
}
```

**Результат (приклад з діапазоном [0.2; 0.8]):**
```
--- Спосіб 1: цикл for ---
   x		   y
-2.0		2.819489
-1.8		2.218498
...
2.0		2.819489

Кількість значень у діапазоні [0.2; 0.8]: 4
```
