## Теоретичні відомості

Метод (функція) — це сукупність команд, що дозволяє виконати деяку операцію в програмі. Методи дозволяють структурувати код, уникати дублювання та підвищувати читабельність програми.

### Оголошення функції в TypeScript

```typescript
function functionName(param1: type1, param2: type2): returnType {
  // тіло функції
  return result;
}
```

### Приклад: функція з параметрами та поверненням значення

```typescript
function add(a: number, b: number): number {
  return a + b;
}

const sum = add(3, 5); // 8
```

### Функція без повернення значення (void)

```typescript
function greet(name: string): void {
  console.log(`Привіт, ${name}!`);
}
```

### Завдання 1 — Обчислення площі трапеції (Варіант 6)

**Файл:** [`src/task1.ts`](src/task1.ts)

Створено функцію `trapezoidArea`, яка приймає дві основи та висоту трапеції і повертає її площу за формулою: `S = ((a + b) / 2) * h`.

```typescript
function trapezoidArea(a: number, b: number, h: number): number {
  return ((a + b) / 2) * h;
}

const a = 5;
const b = 10;
const h = 4;

const area = trapezoidArea(a, b, h);
console.log(`Площа трапеції = ${area}`);
```

```
Основа b = 10

```

### Завдання 2 — Найменше з трьох чисел

**Файл:** [`src/task2.ts`](src/task2.ts)

Створено функцію `findMin`, яка приймає три числа, виводить найменше з них у консоль та повертає його.

```typescript
function findMin(a: number, b: number, c: number): number {
  let min = a;
  if (b < min) {
    min = b;
  }
  if (c < min) {
    min = c;
  }
  console.log(`Найменше число серед ${a}, ${b}, ${c} — це ${min}`);
  return min;
}
```

**Результат:**

### Завдання 3 — Виведення масиву

**Файл:** [`src/task3.ts`](src/task3.ts)

Створено функцію `printArray`, яка приймає масив чисел і виводить його у форматованому вигляді.

```typescript
function printArray(arr: number[]): void {
  let result = "[";
  for (let i = 0; i < arr.length; i++) {
    result += arr[i];
    if (i < arr.length - 1) {
      result += ", ";
    }
  }
  result += "]";
  console.log(result);
}
```

**Результат:**

### Завдання 4 — Найбільше число з масиву

**Файл:** [`src/task4.ts`](src/task4.ts)

Створено функцію `findMax`, яка приймає масив чисел і повертає найбільший елемент.

```typescript
function findMax(arr: number[]): number {
  let max = arr[0]!;
  for (let i = 1; i < arr.length; i++) {
    if (arr[i]! > max) {
      max = arr[i]!;
    }
  }
  return max;
}
```

**Результат:**
```
--- Завдання 4: Найбільше число з масиву ---

Найбільше число: 42

Масив: [-10, -3, -25, -1, -7]
Найбільше число: -1

Масив: [100, 200, 300, 50, 150]
Найбільше число: 300
```
